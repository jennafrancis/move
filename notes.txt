Yelp for fitness studios AKA Move.

A move is a group fitness class.

Models:
- User
  (handled by Devise, omniauth Facebook)
    has_many :reviews
    has_many :group_classes, through: reviews
    * method for avg rating, # of reviews, favorite class

- Studio
    - name
    - street address
    - city_id
    belongs_to :city
    has_many :group_classes
    has_many :reviews, through: :group_classes
    * method for avg rating, # of reviews, most popular class

- GroupClass
    - name
    - instructor
    - studio_id
    belongs_to :studio
    has_many :reviews
    has_many :category_group_classes
    has_many :categories, through: :category_group_classes
    * method for avg rating, # of reviews
    * nest routes within studio

- City
    - name
    has_many :studios

- Review
    - content
    - rating
    - user_id
    - group_class_id
    belongs_to :user
    belongs_to :group_class
    * nested form for optional group_class addition

- CategoryGroupClass
    - category_id
    - group_class_id
    belongs_to :category
    belongs_to :group_class

- Category
    - name
    has_many :category_group_classes
    has_many :group_classes, through: :category_group_classes

** validate within reason, show error messages/nav in layout
